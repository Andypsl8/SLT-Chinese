# 这是什么资源包？
*扫雷艇中文*是一款适用于Minecraft Java版的语言资源包。此包可以~~让你见识到真正的西式中文~~极大地破坏游戏文本的可读性，快试试吧。
# 适用于什么版本？
此包针对Java版**1.16.5**所设计，未来可能会适配更高版本，在其他版本上的可用性尚不明确。
